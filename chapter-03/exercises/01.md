**Q**: When reading or writing to a disk file, are the functions described in this
chapter really unbuffered? Explain.

---

**A**: No, the functions in this chapter are not really unbuffered. When we write
data to a file, the data is copied into a kernel buffer and queued for writing
to disk at some later time. When reading, the system tries to read in more data
than an application requests, thus, improving performance.
